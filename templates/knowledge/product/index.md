## Naming Convention

Feature docs are named `{feature-slug}.md` (e.g., `user-auth.md`, `billing.md`). Each file contains: overview, requirements, user stories, constraints, and current status. Mark deprecated requirements with `[DEPRECATED]` — never delete them.
